Для установки MGR необходимо выполнить:
1. Скачать архив с инсталлятором `publish.rar` из репозитория.
2. Распаковать архив в удобное место.
3. Запустить файл `setup.exe` из распакованной папки.
4. Следуйте инструкциям установщика, чтобы завершить установку.

После всех этапов на рабочем столе должен появиться ярлык для запуска приложения.
